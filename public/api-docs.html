eSIM Myanmar API Documentation

GET /api/coverage

Retrieve 5G coverage maps for Myanmar and ASEAN regions

https://esim.com.mm/api/coverage?region=myanmar
GET /api/speed-test

Initiate network speed testing for connected devices

https://esim.com.mm/api/speed-test
POST /api/activate

Activate eSIM profile via QR code with Microsoft Graph integration

POST /api/activate { "qr_code": "string", "device_id": "string" }
GET /api/devices

List compatible devices and eSIM support status

https://esim.com.mm/api/devices
GET /api/news

Latest news and updates from eSIM Myanmar

https://esim.com.mm/api/news?lang=en
